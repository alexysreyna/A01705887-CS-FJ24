GET: Se usa para leer o recuperar un recurso, por lo que la salida es la información solicitada. HEAD: Se usa muy similar a GET, con la diferencia de que no se puede introducir texto. POST: Se utiliza para crear un nuevo recurso, por lo que se deben definir los datos de la entidad a crear. PUT: Se usa PUT para modificar un recurso, si no hay un recurso de ese tipo, entonces se crea uno nuevo. PATCH: Se utiliza PATCH para modificar una parte de un recurso. DELETE: Se usa para borrar un recurso. ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?
